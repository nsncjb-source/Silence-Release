# 🔇 SILENCE V2 - Discord Nuke Bot

**"Others fight Discord. We just go with the flow."**

---

## ⚡ Features

| # | Command | What It Does |
|---|---------|--------------|
| 1 | Nuke | Delete all channels and roles |
| 2 | Create Channels | Mass create text channels |
| 3 | Spam Channels | Send messages to all channels |
| 4 | Webhook Spam | Create webhooks and spam |
| 5 | Kick All | Kick every member |
| 6 | Ban All | Ban every member |
| 7 | Create Roles | Mass create roles |
| 8 | Get Admin | Give everyone admin |
| 9 | Change Server | Rename the server |
| 10 | DM All | DM every member |
| 11 | Auto Raid | Nuke + Create + Spam in one |
| 12 | Schedule Raid | Set timer for auto raid |
| 13 | Proxy Manager | Load proxies for stealth |
| 14 | Save Config | Save current settings |
| 15 | Load Config | Load saved settings |
| 16 | Multi Server | Raid multiple servers |
| 17 | Stats | View raid statistics |
| 18 | Stealth Mode | Toggle stealth mode |
| 19 | Token Checker | Check if token is valid |
| 20 | Server Info | Get server information |
| 21 | Help | Show all commands |
| 22 | Exit | Close the bot |

---

## 📥 Installation

1. **Download** `SilenceV2.exe` from releases
2. **Run** the exe file
3. **Enter** your bot token
4. **Enter** the server ID
5. **Choose** your command
